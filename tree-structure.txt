═══════════════════════════════════════════════════════════════════════════════

🔄 FLUJO DE UNA SOLICITUD
═══════════════════════════════════════════════════════════════════════════════

1. Usuario abre http://localhost/
   ↓
2. Solicitud cae en API Gateway (Nginx)
   ↓
3. Gateway enruta a Frontend React
   ↓
4. React carga y hace fetch a /api/products
   ↓
5. Gateway enruta /api/products a Products Service
   ↓
6. Products Service consulta PostgreSQL
   ↓
7. Retorna JSON al Gateway
   ↓
8. Gateway aplica compresión y headers de seguridad
   ↓
9. React renderiza componentes con datos
   ↓
10. Usuario ve tienda reactiva en tiempo real

⏱️ Tiempo total: < 200ms
